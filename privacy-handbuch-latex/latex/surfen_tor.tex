\subsection{Tor für Debian und Ubuntu}
Einige Linux-Distributionen enthalten Tor, Vidalia und TorButton. Die Komponenten können mit dem Paketmanger der Distribution installiert werden. Für Debian und Ubuntu spült aptitude alles nötige auf die Platte:
\begin{verbatim}
  # aptitude install tor vidalia xul-ext-torbutton
\end{verbatim}

In der Menügruppe Internet findet man nach der Installation \textit{Vidalia (Tor GUI)}. Über dieses Interface kann Tor bei Bedarf gestartet und gestoppt werden.
